# AGENTS.md

Purpose: Guidance for anyone (human or AI) working in this repo.

## Principles
- Keep changes scoped and explicit.
- Prefer clarity over cleverness.
- Avoid breaking working pages without approval.

## Workflow
- Read the current repo structure before editing.
- Make small, reviewable commits.
- Update or add tests if behavior changes.

## Style
- Default to ASCII in source files.
- Keep UI copy consistent across ES/EN.
- Reuse existing components and patterns.

## When unsure
- Ask a quick question before making assumptions.
